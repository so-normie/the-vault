#card 

- **Solution from Attempt #1**: Use `CONCAT` and `LEFT`

	```sql
	/* Write your T-SQL query statement below */
	SELECT
	    p.person_id AS 'person_id',
	    CONCAT(p.name,'(', LEFT(p.profession, 1),')') AS 'name'
	FROM Person p
	ORDER BY person_id DESC;
	```
