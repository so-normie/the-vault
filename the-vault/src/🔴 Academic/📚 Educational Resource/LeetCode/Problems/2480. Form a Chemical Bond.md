- **Solution from Attempt #1**: Simply do an `INNER JOIN`

	```sql
	/* Write your T-SQL query statement below */
	SELECT
	    e1.symbol AS 'metal',
	    e2.symbol AS 'nonmetal'
	FROM 
	    Elements e1
	INNER JOIN 
	    Elements e2
	    ON 
	        e1.type = 'Metal' AND
	        e2.type = 'Nonmetal'
	```
